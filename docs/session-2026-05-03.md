- All grabbers pass `max_age_days` to `fetch_cached`

### Unified scan-all.py
- `scan-all.py`: budget-based orchestrator (default 100 fetches)
- 50% budget for grabbing new maps (sources sorted by staleness), 50% for URL-checking existing maps
- Supports `--budget N`, `--source <id>`, `--dry-run`
- Sources prioritized by overdue ratio (staleness / interval)
- Writes scan history to `index/scan-history.json` (last 10 entries)
- Dedicated `scan-history.html` page shows all scans with per-source deltas (new/kept/removed)
- Link to scan history in main page infobar
