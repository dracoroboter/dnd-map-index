dnd-map-index

Free D&D 5e battle map index — GitHub
About this project. A personal, open-source (GPL2) index of free D&D 5e battle maps with structured metadata. This is a hobby project with limited scope — not a complete service. For a professional alternative with 5000+ maps, see Lost Atlas.
Maps are not hosted here — the index contains only metadata and links to the original sources. All maps remain property of their respective authors.
Full documentation: README.
